# telegram-bot-java #
A Java API library based on Telegram Bot API to create telegram bots

### Maven Dependency for Consumers ###
Add to your pom.xml:

```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>telegram-bot-java</artifactId>
	<version>1.0.2</version>
</dependency>
```
